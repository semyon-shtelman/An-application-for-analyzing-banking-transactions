# Приложение для анализа банковских транзакций

Веб-приложение для анализа и визуализации банковских транзакций с поддержкой кешбэка, курсов валют и котировок акций.

##  Функциональность

### Основные возможности:
- **Главная страница** с общей статистикой
- **Анализ транзакций** по категориям и периодам
- **Кешбэк** по картам и категориям
- **Курсы валют** в реальном времени
- **Котировки акций** S&P500
- **Фильтрация данных** по датам

### Страницы приложения:
1. **Главная** - обзор финансовой активности
2. **Отчеты** - детализированные отчеты по категориям
3. **Настройки** - пользовательские настройки


##  Установка и запуск

### 1. Клонирование репозитория
```
git clone <repository-url>
cd An-application-for-analyzing-banking-transactions
```
### Настройка переменных окружения
Создайте файл .env и добавьте:
API_KEY=your_alpha_vantage_api_key

## Примеры использования
Главная страница
Возвращает:
- Приветствие (по времени суток)
- Информация по картам
- Топ-5 транзакций
- Курсы валют
- Котировки акций

```
result = generate_main_response('2021-12-30 12:30:00')
```
Анализ трат по категориям
```
from src.reports import spending_by_category

# Получить траты по категории "Еда" за последние 3 месяца
result = spending_by_category(df, "Еда", "2024-03-31")
```
Получение кэшбэка
```
from src.services import anylize_cashback

# Проанализировать кешбэк по категориям за январь 2024
cashback_data = anylize_cashback(df, 2024, 1)
```
## Логирование
Приложение использует многоуровневое логирование:

- INFO: Основные операции
- DEBUG: Детальная отладка
- WARNING: Предупреждения
- ERROR: Ошибки выполнения
